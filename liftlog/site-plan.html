Site Purpose

LiftLog is a personal fitness web application designed to help users organize workouts, stay consistent, and track their progress in a simple and motivating way. The site will make it easier for users to choose exercises, record workout details, and view progress so they can stay focused on their fitness goals.

Target Audience

The intended audience is teens and adults who want a simple way to plan and track workouts without using a complicated fitness app. It is especially helpful for beginners, busy parents, and gym users who want a clean and easy to use tool for strength training and exercise planning.

Description of Dynamic Elements

JavaScript will be used to make the site interactive and useful. Users will be able to add workouts, select exercises from categories, and save workout data using localStorage so their progress stays on the site even after the page is refreshed.

The site will also include dynamic filtering so users can sort exercises by muscle group or workout type. A progress section will display totals such as number of workouts completed, favorite exercise categories, and recent activity. Buttons, form validation, and interactive cards will help create a more engaging user experience.

Content

Home Page

Hero heading: Build Strength. Track Progress. Stay Consistent.

Hero text: LiftLog helps you organize workouts, log exercises, and stay motivated with a simple fitness tracker built for real life.

Call to action buttons: Start Tracking | View Exercises

Features Section

  • Track daily workouts and exercise details
  • Browse exercises by muscle group
  • Save your progress directly in the browser
  • Stay motivated with simple visual progress updates

Exercise Library Page

This page will include exercise cards with exercise name, target muscle group, equipment needed, and suggested sets and reps.

Sample exercises:

  • Squat — Legs — Barbell — 4 sets of 8
  • Bench Press — Chest — Barbell — 4 sets of 8
  • Lat Pulldown — Back — Cable Machine — 3 sets of 10
  • Shoulder Press — Shoulders — Dumbbells — 3 sets of 10
  • Romanian Deadlift — Hamstrings — Barbell — 4 sets of 8
  • Bicep Curl — Arms — Dumbbells — 3 sets of 12

Workout Log Page

This page will allow users to enter workout date, exercise name, sets, reps, and notes. The page will display saved workouts and allow users to remove or update entries.

Images and Graphics Needed

  • Fitness themed hero image
  • Simple icons for workouts, progress, and exercise categories
